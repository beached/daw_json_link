template<json_options_t PolicyFlags = json_details::default_policy_flag, typename Allocator = json_details::NoAllocator>
struct daw::json::anonymous_namespace{daw_json_parse_policy.h}::BasicParsePolicy< PolicyFlags, Allocator >

Handles the bounds and policy items for parsing execution and comments.

Template Parameters
PolicyFlagsset via parse_options method to change compile time parser options
AllocatorAn optional Allocator to allow for passing to objects created while parsing if they support the Allocator protocol of either the Allocator argument being last or with a first argument of std::allocator_arg_t followed by the allocator.Thing( args..., alloc ) or Thing( std::allocator_arg, alloc, args... )
